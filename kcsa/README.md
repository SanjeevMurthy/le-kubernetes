# KCSA - Kubernetes and Cloud Native Security Associate

## Exam Format

- Multiple choice (not performance-based)
- 60 questions, 90 minutes
- Passing score: 75%

## Exam Domains & Weights (2025)

| Domain | Weight |
|--------|--------|
| Overview of Cloud Native Security | 14% |
| Kubernetes Cluster Component Security | 22% |
| Kubernetes Security Fundamentals | 22% |
| Kubernetes Threat Model | 16% |
| Platform Security | 16% |
| Compliance and Security Frameworks | 10% |

## Key Topics

- **Cloud Native Security**: 4C's of security (Cloud, Cluster, Container, Code), attack surfaces
- **Cluster Components**: API server security, etcd encryption, kubelet authentication
- **Security Fundamentals**: Pod security standards, RBAC, network policies, secrets management
- **Threat Model**: STRIDE, MITRE ATT&CK for containers, common attack vectors
- **Platform Security**: Supply chain, image signing, admission controllers
- **Compliance**: CIS benchmarks, SOC 2, NIST frameworks
