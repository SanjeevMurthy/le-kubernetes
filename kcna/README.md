# KCNA - Kubernetes and Cloud Native Associate

## Exam Format

- Multiple choice (not performance-based)
- 60 questions, 90 minutes
- Passing score: 75%

## Exam Domains & Weights (2025)

| Domain | Weight |
|--------|--------|
| Kubernetes Fundamentals | 46% |
| Container Orchestration | 22% |
| Cloud Native Architecture | 16% |
| Cloud Native Observability | 8% |
| Cloud Native Application Delivery | 8% |

## Key Topics

- **K8s Fundamentals**: Architecture, API primitives, containers, scheduling, RBAC
- **Container Orchestration**: Container networking, service mesh, storage
- **Cloud Native Architecture**: Microservices, serverless, CNCF landscape
- **Observability**: Prometheus, Grafana, tracing, logging
- **Application Delivery**: GitOps, CI/CD, Helm, Kustomize
