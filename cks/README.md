# CKS - Certified Kubernetes Security Specialist

## Exam Domains & Weights (2025)

| Domain | Weight |
|--------|--------|
| Cluster Setup | 10% |
| Cluster Hardening | 15% |
| System Hardening | 15% |
| Minimize Microservice Vulnerabilities | 20% |
| Supply Chain Security | 20% |
| Monitoring, Logging and Runtime Security | 20% |

## Key Topics

- **Cluster Setup**: Network policies, CIS benchmarks, Ingress with TLS, node metadata protection
- **Cluster Hardening**: RBAC, service accounts, restrict API access, upgrade Kubernetes
- **System Hardening**: OS-level security, IAM, AppArmor, Seccomp
- **Microservice Vulnerabilities**: PSA/PSS, OPA Gatekeeper, Secrets management, runtime sandboxes (gVisor)
- **Supply Chain Security**: Image scanning (Trivy), ImagePolicyWebhook, allowlisting registries, static analysis
- **Runtime Security**: Falco, audit logs, immutable containers, behavioral analytics

## Prerequisites

- CKA certification (required before taking CKS)
